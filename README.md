<div class="research-section">
  <h3 class="research-title">1.柑橘养分高效利用</h3>
  <p class="research-content">
   针对柑橘养分利用率低的问题，利用丰富的砧木/砧穗种质资源筛选养分高效利用的砧木品种或砧穗组合，基于根际生命共同体的理论，阐明砧木/砧穗-微生物互作提高养分利用的机制，研发微生物肥料及增效产品，促进柑橘产业绿色高质量发展。<br>
</p>
</div>
<div class="research-section">
  <h3 class="research-title">2.养分供应与品质形成</h3>
  <p class="research-content">
  分析不同养分水平、配比和施用方式对作物产量、外观品质、营养品质与风味品质的影响。
  </p>
</div>
<div class="research-section">
  <h3 class="research-title">3.柑橘轻简化施肥技术和产品研发</h3>
  <p class="research-content">
  为满足未来智慧化果园的需求，研发主要柑橘品种的肥水一体化精准施肥技术方案，配套并研发相应水溶性肥料、肥料增效等产品；为解决产区钙镁严重缺乏的问题，研发相应的缓释肥料、螯合肥料等产品，为产业绿色高质量发展提供技术和产品。
   </p>
</div>
<div class="research-section">
  <h3 class="research-title">4.逆境胁迫下的养分利用</h3>
  <p class="research-content">
  关注干旱、盐碱、低温等逆境对植物养分吸收与代谢的影响，探索适应性调控机制。
   </p>
</div>
<div class="research-section">
  <h3 class="research-title">5.绿色高效栽培技术</h3>
  <p class="research-content">
  结合田间试验和实验室研究，构建精准施肥、绿色栽培和品质协同提升技术体系。
   </p>
</div>
